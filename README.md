# Bit Vitals — M0 プレイアブル・モック

「**bank or push（確定するか、欲張るか）**」が面白いかを白い図形で確かめる、単一HTML（Canvas+JS）の遊べる検証用モック。**本番ではありません**（本番は Godot で実装予定）。

▶ **プレイ:** https://ryogaaaaaaaaaaa.github.io/bit-vital/

## 遊び方
- **スマホ**: 画面左を**ドラッグ**で移動 / 右下 **DASH**（タップ）/ **EXTRACT**（長押しで脱出）
- **PC**: 移動 `WASD`・矢印 / ダッシュ `Space`・`Shift` / 脱出 `E` 長押し / 開始・再戦 `Space`
- 攻撃は**自動**（最近接の敵を撃つ）。狙う必要なし。

## ルール（＝検証したい核）
1. 敵を倒すと素材が落ちる。拾うと **UNBANKED**（未確定）が増える。
2. **EXTRACT 長押し**（約3秒）で脱出 → 成功すると UNBANKED を**確定**。
3. **死ぬと UNBANKED の大半（80%）を失う。**
4. 時間とレア回収で **Risk** が上がり、敵が濃く・強くなる。Risk4 で **HUNTER**（脱出妨害の追跡敵）が出現。
5. → 常に「いま確定するか、もう1個レアを狙うか」を迫られる。

## メモ
- 手触りの調整は `index.html` 冒頭の `CFG`（`riskTimeToMax` / `channelTime` / `lossRatio` / `spawnBase` など）。
- 見るポイント: 「もう1回」と指が伸びるか。「あと1個」を狙って死ぬ瞬間に笑えるか。
