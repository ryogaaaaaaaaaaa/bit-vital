# Turf Commander（陣取りコマンダー・仮） — 白図形プロト

3分のリアルタイム陣取り。**メイン1機を操作しつつ、サブ2機に方針を出す“指揮官”**が遊びの核。白い図形の一次プロト（本番ではない／テーマも未定）。

▶ **プレイ:** https://ryogaaaaaaaaaaa.github.io/bit-vital/turf/

## 操作
- **タップ**で開始 / **左ドラッグ**＝メイン機を移動
- 上の **SUB1・SUB2 ピルをタップで方針切替**（回収→護衛→妨害→防衛）＝エッセンス
- 右下 **PLACE**＝自陣に隣接するマスにメインがいる時、◆パーツを使って確保（押せる時だけ青く点灯）
- PC: `WASD`/矢印で移動、`Space`でPLACE、`1`/`2`で方針切替

## ルール
- 陣地パーツがランダム落下 → **全機が回収**して共有プールへ → **置けるのはメインだけ**。
- 全機にHP、撃破で数秒リスポーン（→ 敵メインを妨害で潰すと相手の拡張が止まる）。
- 敵を線で**囲むと内側を一括制圧**。
- **3分**で 面積が大きい方 or 敵を全滅させた方の勝ち。

## 見るポイント
1. 1機操作＋2機采配の「指揮官」感は楽しい？
2. *置けるのはメインだけ* の駆け引き（敵メイン狙い）は効いてる？
3. 包囲が決まった瞬間は気持ちいい？
